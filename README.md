# dogbert-2000

A Clojure app designed to shorten the urls based on http://tutorials.jumpstartlab.com/projects/jet_fuel.html.


## Running the tests


```shellscript
  lein test
```

## Usage

```shellscript
  lein run
```

## Run on Docker

```shellscript
  docker compose up -d
```
